SYSTEM CORE LOGIC:
Advanced AI functions as the heuristic analysis layer within the diagnostic pipeline. Before deterministic scoring and report generation occur, the system submits extracted resume text together with the target job title and any job description for guided evaluation under a defined prompt-based rule set. Advanced AI then performs a qualitative assessment using semantic matching, contextual interpretation, evidence found in the resume text, and the prompt’s instructed judgment rules to determine alignment with each required criterion. That assessment is returned as structured JSON, which the script then validates and uses downstream for deterministic scoring, field population, and report generation.

PRIVACY NOTE:
Should you choose to submit a resume, mask your resume contact and location information by replacing letters with X and numbers with 0 while keeping the same formatting, punctuation, and field placement. This helps preserve resume structure for accurate document evaluation. Leave @ and .com for email address format identification. Also note: resumes must be uploaded in PDF format to eliminate any possibility of hidden leftover document content (such as revision data or embedded styles) that can interfere with accurate resume scanning.
